# OCCPGameServer

The Open Cyber Challenge Platform primary game engine.

## Installation

Add this line to your application's Gemfile:

    gem 'OCCPGameServer'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install OCCPGameServer

## Usage

    $ occpgs -f instance.xml
